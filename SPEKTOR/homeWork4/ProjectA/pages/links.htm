🔗 Links in HTML

📎 Relative URL

Refers to the file's location relative to the current page.

This means:
Go one folder up (../)
Then look for index.html

📌 Example: a href="../homePage.html

🌍 Absolute URL

A full URL, including the domain or full file path.

📌 Example: a href="file:///Users/spektor/Desktop/John Bryce/Workspace/homeWork4/ProjectA/homePage.html

❌ Browsers like Chrome block file:/// links when clicked from a page — for security.
